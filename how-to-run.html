How to Run the Interactive Resume Game

Important: This game must be run from a web server to avoid CORS errors when loading assets. Opening the HTML file directly in a browser will not work properly.

Option 1: Using the included Node.js server (Recommended)

  1. Make sure you have Node.js installed on your computer.
  2. Open a command prompt or terminal in the game directory.
  3. Run the following command: 
    node server.js
  4. Open your web browser and navigate to: http://localhost:3000
This is the recommended method as it uses the custom server that's specifically configured for this game.

Option 2: Using another web server

If you prefer, you can use any of these alternative methods:

Using Node.js http-server

# Install http-server globally (if not already installed)
npm install -g http-server

# Run the server in the game directory
http-server

# Then open http://localhost:8080 in your browser

Using Python's built-in server

# For Python 3
python -m http.server

# For Python 2
python -m SimpleHTTPServer

# Then open http://localhost:8000 in your browser

Using VS Code Live Server

If you're using Visual Studio Code:

  1. Install the "Live Server" extension
  2. Right-click on index.html and select "Open with Live Server"
  3. The game will open in your default browser

Troubleshooting

Common Issues:

  • CORS Errors: If you see CORS errors in the console, make sure you're running the game through a web server and not opening the HTML file directly.
  • Audio not playing: Click anywhere on the game to enable audio. Browsers require user interaction before allowing audio to play.
  • Missing assets: Some placeholder assets are used if the actual assets are not found. This is normal and won't affect the core functionality.

For more information, please refer to the README.md file.
